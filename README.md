# Retail Sales Analysis - Real World Data Project

## 📌 Overview

This project analyzes retail sales data to uncover customer purchasing behavior, sales trends, product performance, and business insights. The goal is to perform end-to-end data analysis on a real-world dataset and present actionable recommendations.

## 🎯 Objectives

- Analyze sales performance
- Identify top-selling products
- Understand customer purchasing patterns
- Track revenue trends over time
- Generate business insights using visualizations

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
